Problem Statement
Given integers A, X, and M, find ∑_i = 0^X-1 A^i, modulo M.

Constraints

1 ≤A, M ≤10^9
1 ≤X ≤10^12
All values in the input are integers.


Input
The input is given from Standard Input
 in the following format:
A X M


Output
Print the answer.

Sample Input

 13 4 7


Sample Output

 15

3^0 + 3^1 + 3^2 + 3^3 = 40, which equals 5 modulo 7, so 5 should be printed.

Sample Input

 28 10 9


Sample Output

 20


Sample Input

 31000000000 1000000000000 998244353


Sample Output

 3919667211
